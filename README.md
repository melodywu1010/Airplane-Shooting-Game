# 天際防衛戰

使用 **Pygame** 製作的 2D 直向射擊小遊戲。玩家操控戰鬥機在畫面中移動（**自動射擊**），在**限時**內完成 5 個關卡。第 3 關為中 Boss、第 5 關為大 Boss；戰鬥中可拾取 **技能果實** 強化能力。

---

## 專題簡介

- 類型：直向捲軸射擊（Arcade Shooter）
- 關卡：共 5 關（含中 Boss 與大 Boss）
- 特色：
  - 自動射擊（不需按鍵）
  - 多種敵人造型
  - 戰鬥機尾焰閃爍動畫
  - 限時過關機制
  - 技能果實（回血/加速射擊/散射）與 Boss 固定掉落

---

## 使用套件 / 環境

- Python 3.x
- pygame
